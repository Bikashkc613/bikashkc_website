## 🐛 Troubleshooting

### **Problem: Sections Don't Appear**
**Solution:**
- Clear browser cache (Ctrl + F5)
- Check console for errors (F12)
- Ensure all CSS files are loaded

### **Problem: Filtering Doesn't Work**
**Solution:**
- Check js/main.js is loaded
- Verify project cards have `data-category` attribute
- Check browser console for JavaScript errors

### **Problem: Timeline Doesn't Display**
**Solution:**
- Ensure timeline CSS is loaded
- Check for missing closing tags
- Verify AOS animations are working

### **Problem: Images Not Loading**
**Solution:**
- Check image paths are correct
- Images using Unsplash placeholder URLs need internet
- Replace with your own images
