4. **Customize content** in `index.html`

6. **Deploy** to GitHub Pages, Vercel, or Netlify

## 🌐 Deployment

### GitHub Pages
1. Push to GitHub repository
2. Go to Settings → Pages
3. Select branch and save
4. Access at `username.github.io/repository`

### Vercel
