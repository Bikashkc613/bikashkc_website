# 📄 CV & Resume Templates Guide

## How to Create Professional Templates

### Option 1: Use Online Template Makers (Easiest)

#### **Canva (Free & Easy)**
1. Go to [canva.com](https://canva.com)
2. Search for "CV Template" or "Resume Template"
3. Choose a professional template
4. Customize with your information
5. Download as:
   - **PDF** for CV.pdf and Resume.pdf
   - **DOCX** for CV_Template.docx and Resume_Template.docx

#### **Microsoft Word (Free Templates)**
1. Open Microsoft Word
2. Go to **File → New**
3. Search "CV" or "Resume"
4. Choose professional template
5. Customize and save as:
   - **PDF** (File → Save As → PDF)
   - **DOCX** (File → Save As → Word Document)

#### **Google Docs (Free)**
1. Go to [docs.google.com/templates](https://docs.google.com/templates)
2. Search "Resume" or "CV"
3. Select template
4. Edit with your info
5. Download:
   - **PDF**: File → Download → PDF
   - **DOCX**: File → Download → Microsoft Word

## 🎯 Best Practices

### **For PDF Files (CV.pdf, Resume.pdf)**

✅ **DO:**
- Use professional fonts (Inter, Arial, Calibri)
- Keep file size under 1MB
- Use consistent formatting
- Include contact information
- Add your portfolio URL
- Use bullet points for clarity
- Include relevant keywords

❌ **DON'T:**
- Use fancy fonts
- Include photos (unless required)
- Use bright colors
- Make it too long (2 pages max)
- Include personal details (age, etc.)

### **For DOCX Templates (Editable)**

✅ **DO:**
- Use Word styles for consistency
- Include placeholder text like [Your Name]
- Add comments with instructions
- Use tables for layout
- Keep formatting simple
- Make sections clearly marked
- Include all necessary sections

❌ **DON'T:**
- Lock the document
- Use complex formatting
- Hide important sections
- Make it too complicated
- Use password protection

## 🚀 Quick Template Creation Process

### **Step 1: Create CV (15 minutes)**
1. Open Word/Google Docs
2. Choose CV template
3. Add your information
4. Save as **CV.pdf**
5. File → Save As → Change to DOCX → Save as **CV_Template.docx**
6. In template, replace your info with placeholders:
   - "Bijay KC" → "[Your Name]"
   - "your-email@example.com" → "[Your Email]"
   - Add instructions in brackets

### **Step 2: Create Resume (10 minutes)**
1. Use shorter, 1-page format
2. Focus on key achievements
3. Save as **Resume.pdf**
4. Save template version as **Resume_Template.docx**
5. Add placeholders like CV template

### **Step 3: Upload to Portfolio (2 minutes)**
1. Place all 4 files in `assets/` folder:
   ```
   assets/
   ├── CV.pdf
   ├── Resume.pdf
   ├── CV_Template.docx
   └── Resume_Template.docx
   ```
2. Test downloads on your website

## 🎨 Design Tips

### **Color Scheme (Match Your Portfolio)**
- Primary: #00E5FF (Cyan)
- Secondary: #6C63FF (Purple)
- Accent: #14F195 (Green)
- Text: Dark gray or black
- Background: White

### **Typography**
- **Headers**: 16-18pt, Bold
- **Body**: 10-12pt, Regular
- **Name**: 24-28pt, Bold
- **Section titles**: 14-16pt, Bold

### **Layout**
- **Margins**: 0.5-1 inch all sides
- **Line spacing**: 1.15 or 1.5
- **Sections**: Clear separation
- **White space**: Don't overcrowd

## 🐛 Troubleshooting Downloads

### **Download Not Working**

**Problem**: Button clicks but nothing downloads

**Solutions:**
1. Check file is in `assets/` folder
2. Check file name matches exactly
3. File path should be: `assets/CV.pdf` (not `assets/assets/CV.pdf`)
4. Clear browser cache
5. Try different browser

### **Wrong File Downloads**

**Problem**: Clicking CV downloads Resume

**Solution:**
- Check HTML links:
```html
<!-- CV Button -->
<a href="assets/CV.pdf" download>

<!-- Resume Button -->  
<a href="assets/Resume.pdf" download>

<!-- CV Template -->
<a href="assets/CV_Template.docx" download>

<!-- Resume Template -->
<a href="assets/Resume_Template.docx" download>
```

### **File Opens Instead of Downloads**

**Problem**: PDF opens in browser instead of downloading

**Solution:**
- This is normal browser behavior
- User can still save it
- Or add `download="filename.pdf"` attribute:
```html
<a href="assets/CV.pdf" download="Bijay-KC-CV.pdf">
```
