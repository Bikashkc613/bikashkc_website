# Assets Folder

Place your files here:

1. **CV.pdf** - Your Curriculum Vitae
2. **Resume.pdf** - Your Resume
3. **Resume_Template.docx** - Resume Template for download
4. **CV_Template.docx** - CV Template for download
5. **favicon.png** - Website favicon (32x32 or 16x16 pixels)

## File Requirements

### CV.pdf & Resume.pdf
- Format: PDF
- Max size: 2MB recommended
- Professional formatting
- Updated contact information

### Resume_Template.docx & CV_Template.docx
- Format: Microsoft Word (.docx)
- Editable templates
- Professional layouts
- Easy to customize

### favicon.png
- Format: PNG
- Size: 32x32px or 16x16px
- Transparent background recommended
- Simple, recognizable icon

## Note
The download links in the portfolio will automatically download these files when clicked.
