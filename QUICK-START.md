# ⚡ Quick Start Guide - 5 Minutes to Launch

## 🎯 Fastest Way to Get Started

### Step 1: Update Personal Info (2 minutes)

Open `index.html` and find/replace:

```javascript
// Find → Replace with your info
"Bijay KC" → "Your Name"
"kcbikash200030004000@gmail.com" → "your@email.com"
"9761550629" → "Your Phone"
"Sandhikharka-9, Arghakhanchi" → "Your Location"
```

### Step 2: Add Your Photos (1 minute)

1. Add profile photo to `images/` folder
2. Name it: `profile.jpg` (or update path in HTML)
3. Recommended size: 800x800px

### Step 3: Add Documents (1 minute)

Add to `assets/` folder:
- `CV.pdf` - Your CV
- `Resume.pdf` - Your Resume  
- `Resume_Template.docx` - Template
- `favicon.png` - Website icon

### Step 4: Update Links (30 seconds)

Update social media URLs in HTML:
```html
<a href="https://github.com/YourUsername">
<a href="https://linkedin.com/in/YourProfile">
<a href="https://facebook.com/YourProfile">
```

### Step 5: Test & Deploy (30 seconds)

1. Open `index.html` in browser
2. Test all links
3. Deploy to GitHub Pages, Vercel, or Netlify

## 🎨 Quick Customizations

### Change Colors
Edit `css/style.css` line 6-8:
```css
--accent-primary: #00E5FF;    /* Your color */
--accent-secondary: #6C63FF;   /* Your color */
--accent-tertiary: #14F195;    /* Your color */
```

### Update Typing Animation
Edit `index.html` line ~130:
```html
data-texts="Title 1|Title 2|Title 3|Title 4"
```

### Update Statistics
Find counter elements and change `data-count`:
```html
<h4 class="counter" data-count="50">0</h4>
```

## 🚀 Deploy in 1 Minute

### GitHub Pages (Easiest)
```bash
git init
git add .
git commit -m "My portfolio"
git branch -M main
git remote add origin https://github.com/username/portfolio.git
git push -u origin main

# Then: Repository Settings → Pages → Enable
```

### Vercel (Fastest)
1. Go to vercel.com
2. Sign in with GitHub
3. Import your repository
4. Click Deploy
5. Done! 🎉

### Netlify (Drag & Drop)
1. Go to netlify.com  
2. Drag your project folder
3. Drop it on the page
4. Done! 🎉

## 🆘 Common Issues & Fixes

### Images Not Showing
```
Fix: Check file path is correct
Fix: Ensure image is in images/ folder
Fix: Clear browser cache (Ctrl + F5)
```

### Downloads Not Working
```
Fix: Ensure PDFs are in assets/ folder
Fix: Check file names match exactly
Fix: Use correct path: assets/CV.pdf
```

### Animations Not Working
```
Fix: Check if all CSS files loaded
Fix: Check if all JS files loaded
Fix: Open console (F12) for errors
```

### Mobile Menu Not Working
```
Fix: Ensure main.js is loaded
Fix: Check for JavaScript errors
Fix: Test in different browser
```

## 💡 Pro Tips

### 1. Optimize Images First
- Use TinyPNG.com before uploading
- Keep under 500KB each
- Use JPG for photos, PNG for logos

### 2. Test on Real Devices
- Test on your actual phone
- Ask friends to check on theirs
- Use Chrome DevTools (F12) → Device Toolbar

### 3. Update Regularly
- Add new projects monthly
- Keep skills current
- Update statistics quarterly

### 4. Make It Yours
- Use your own colors
- Add your personality
